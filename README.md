# Prompt Saver Desktop (Tauri v2)

External library file workflow + portable builds.

## Install
```bash
npm install
```

## Run
```bash
npm run tauri dev
```

## Build
```bash
npm run tauri build
```

## Behavior
- External library file by default
- Save overwrites active `.json`
- Export backup defaults to `<LibraryName>.backup.html`
